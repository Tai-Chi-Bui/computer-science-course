# Week 3: Algorithms

## Overview
This week introduces algorithms, the step-by-step procedures for solving problems.

## Key Concepts
- Searching algorithms (linear, binary)
- Sorting algorithms (bubble, selection, insertion)
- Algorithm efficiency and Big O notation
